Authentication

To use our API, you must include your unique API key in the header of every request. You can find your key on your Dashboard.

Include the header as follows:

X-API-Key: YOUR_API_KEY_HERE

Endpoints

Create Screenshot

POST /screenshot

This is the primary endpoint for creating a new screenshot. The request body must be a JSON object containing the URL to capture.

Example Request Body:

{
    "url": "https://www.google.com"
}

Example Response (Success):

{
    "status": "success",
    "output": "Success! Screenshot saved as 'www_google_com.png'..."
}
