API Documentation

Welcome to the official documentation for the Intelligent Screenshot API. Our API is designed to be simple, predictable, and easy to use.

Authentication

All API requests must be authenticated. You must include your unique API key in the request headers.

Header Name: X-API-Key

You can get your key by signing up on our homepage.

Endpoints

Create User

POST /create_user

This endpoint creates a new user and returns a unique API key. You only need to do this once.

Request Body (JSON):

{ "email": "your.email@example.com" }

Success Response (JSON):

{ "email": "your.email@example.com", "api_key": "YOUR_UNIQUE_API_KEY" }

Request a Screenshot

POST /screenshot

This is the main endpoint for requesting a screenshot. Remember to include your API key in the `X-API-Key` header.

Request Body (JSON):

{ "url": "https://www.google.com" }

Success Response (JSON):

{ "status": "success", "output": "..." }
